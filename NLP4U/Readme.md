## NLP4U
+ Your Guide to Natural Language Processing (NLP) !

#### Description

This is a Natural Language Processing (NLP) Based App useful for basic NLP concepts such as follows :

+ Tokenization
+ Stemming
+ Lemmetization
+ N-grams
+ PoS tagging
+ NER extraction

#### Libraies Used
```bash
nltk
streamlit
```

#### Run
```bash
streamlit run app.py
```
